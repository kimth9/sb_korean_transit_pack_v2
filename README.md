<h1>Korean Transit Pack</h1>

<p>If you are looking for the perfect public transit options to accompany the &quot;Korean Map Pack,&quot; this is the must-have download for you!</p>
<h2>1. Included Transit Systems</h2>
<ul>
<li>GTX (Great Train eXpress) based on GTX-A | In-game Code :  <code>[KR] GTX</code></li>
<li>Korean-style Heavy Metro based on Seoul Line 1 | In-game Code :  <code>[KR] Heavy Metro</code></li>
<li>Korean-style Light Rail Transit based on Incheon Line 2 | In-game Code :  <code>[KR] Light Rail Transit</code></li>
<li>Korean-style Tram based on Wirye Line | In-game Code :  <code>[KR] Tram</code></li>
<li>Korean-style BRT based on Sejong BRT | In-game Code :  <code>[KR] BRT</code></li>
</ul>
<p>All vehicle specifications, construction costs, and operating expenses have been meticulously balanced based on authoritative and official data.</p>

<h2>3. Data Sources</h2>
<p>The data used in this pack is derived from the following sources:</p>
<ul>
<li>GTX Data<ul>
<li>GTX-A Preliminary Feasibility Study Report (Korea Development Institute [KDI], 2024)</li>
<li>Western Metropolitan Area Wide-Area Express Railway (GTX-D) Preliminary Feasibility Study Report (KDI, 2025)</li>
</ul>
</li>
<li>Heavy Metro Data<ul>
<li>Statistical Yearbook of Korean Railroads (Ministry of Land, Infrastructure and Transport, 2024)</li>
<li>Shinbundang Line Northwest Extension (Yongsan~Samsong) Preliminary Feasibility Study Report (KDI, 2023)</li>
</ul>
</li>
<li>Light Rail Transit Data<ul>
<li>Incheon Metro Line 2 Rolling Stock Data</li>
<li>Mokdong Line (Sinwol-dong~Dangsan) Preliminary Feasibility Study Report (KDI, 2025)</li>
</ul>
</li>
<li>Tram Data<ul>
<li>Seoul Wirye Line Tram (Upcoming) Rolling Stock Data</li>
<li>Ulsan Subway Line 1 Re-feasibility Study Report (KDI, 2023)</li>
</ul>
</li>
<li>BRT Data<ul>
<li>Hyundai Electric Articulated Bus Rolling Stock Data</li>
<li>Cheongna~Hwagok BRT Preliminary Feasibility Study Report (KDI, 2006)</li>
</ul>
</li>
<li>Consumer Price Index (CPI) from the Bank of Korea Economic Statistics System (ECOS)</li>
</ul>
